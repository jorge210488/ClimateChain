# ML Service Module

Python service responsible for premium estimation based on climate history and forecast signals.

## Planned Components

- `app/api/` endpoint handlers
- `app/models/` trained model loading/inference
- `app/services/` business logic
- `app/schemas/` request/response schemas
- `tests/` unit and integration tests

## Standards

- Python code and variable naming in English.
- Deterministic response schema for backend integration.
- Versioned model artifact and reproducible training flow.
